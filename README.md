# 📘 Algoritmo

Este repositório contém exercícios da disciplina de **Algoritmos em C++**, desenvolvidos como parte dos meus estudos em Análise e Desenvolvimento de Sistemas.

## 📂 Estrutura
- `C03EX4A.CPP` → Exercício 4A do Capitulo 03 do livro de Algoritmo
- (outros arquivos serão adicionados conforme avanço nos estudos)

## 🚀 Objetivo
Praticar lógica de programação e consolidar conceitos de:
- Estruturas de controle (if, else, switch)
- Laços de repetição (for, while, do-while)
- Funções e modularização

## 📚 Referências utilizadas
- **C++23 para Windows** – José Augusto N. G. Manzano  
- **Algoritmos: Lógica para Desenvolvimento de Programação Imperativa de Computadores** – José Augusto N. G. Manzano  

## 🛠️ Como executar
1. Baixe o arquivo `.CPP`.
2. Compile usando um compilador C++ (ex.: g++, Dev-C++, Code::Blocks).
3. Execute o programa no terminal ou IDE.
